# 📊 Bank Customer Churn Predictor

## 🧠 Project Overview
This is a Machine Learning web application that predicts whether a bank customer will leave (churn) or stay. It uses multiple ML models and provides real-time predictions through an interactive Streamlit interface.

---

## 🚀 Features
- Data preprocessing and cleaning
- Encoding categorical variables
- Feature scaling
- Multiple ML models:
  - Naive Bayes
  - Decision Tree
  - Support Vector Machine (SVM)
- Real-time prediction system
- Interactive Streamlit web app

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

---

## ⚙️ How It Works
1. User enters customer details (age, balance, credit score, etc.)
2. Data is preprocessed automatically
3. Trained ML models make predictions
4. Result shows whether customer will churn or not

---

## ▶️ How to Run Locally

### 1. Install dependencies
```bash
pip install -r requirements.txt

###2. Train models
python train.py

3. Run Streamlit app
streamlit run app.py



---
📊 Output
Churn = 1 → Customer will leave ❌
Churn = 0 → Customer will stay ✅
